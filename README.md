# CUSTOMER-behavior-analysis
End-to-end customer behavior analysis using Python, SQL, and Power BI with data cleaning, querying, and visualization.

📊 Customer Behavior Analysis | Python + SQL + Power BI
📌 Project Overview
This project analyzes customer purchasing behavior using Python, SQL, and Power BI to generate business insights.
⚙️ Tools & Technologies
- 🐍 Python (Pandas, NumPy)
- 🗄️ PostgreSQL
- 📊 Power BI
🧹 Data Cleaning & Preprocessing
- Handled missing values
- Converted data types
- Created new features (age groups, purchase frequency)
🗄️ SQL Analysis
- Customer segmentation (new, returning, loyal)
- Purchase trends
- Discount impact
- Product performance
📊 Power BI Dashboard
Includes:
- Total customers, average purchase, review rating
- Revenue and sales by category
- Customer distribution
- Sales by age group
- Interactive filters
📈 Key Insights
- Identified top-performing categories
- Analyzed customer segments
- Evaluated discount impact
- Observed revenue trends
🎯 Conclusion
End-to-end data analysis from cleaning to visualization.
